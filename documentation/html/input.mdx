# Tailwind CSS Input

Easily create inputs with different status and sizes using our components based on Tailwind CSS.
<br/>
Input fields are an essential user interface design element, providing users with the means to enter non-standardized responses. See below our input components examples:
<br/>

### Input Outline 
Use this example to create outline input components.
<br/>
<ComponentCard id="input-outline">
<div class="input-group input-group-outline">
  <label class="form-label">Outline</label>
  <input type="text" class="form-control" />
</div>
</ComponentCard>

```html showLineNumbers
<div class="input-group input-group-outline">
  <label class="form-label">Outline</label>
  <input type="text" class="form-control" />
</div>
```
<br/>
<br/>
