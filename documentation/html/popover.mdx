# Tailwind CSS Popover

Use our Tailwind CSS Popover to portray people or objects. It can be used as a visual identifier for a user profile on your website.

See below our popover components examples.

<br />

<ComponentCard id="popover">
<button className="button button-pink" data-target="popover" type="button" data-ripple-light="true">Popover top</button>
<div className="popover hidden" data-attribute="popover" data-popper-placement="top">
  <div className="popover-title">Gray popover title</div>
  <div className="popover-body p-3">And here's some amazing content. It's very engaging. Right?</div>
  <div className="arrow" data-popper-arrow></div>
</div>
</ComponentCard>
