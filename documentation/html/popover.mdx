# Tailwind CSS Popover

Use our Tailwind CSS Popover to portray people or objects. It can be used as a visual identifier for a user profile on your website.

See below our popover components examples.

<br />

<ComponentDemo id="popover">
  <button
    className="button button-pink"
    data-target="popover"
    type="button"
    data-ripple-light="true"
  >
    Popover top
  </button>
  <div className="popover hidden" data-attribute="popover" data-placement="top">
    <div className="popover-title">Popover title</div>
    <div className="popover-body p-3">
      And here's some amazing content. It's very engaging. Right?
    </div>
    <div className="arrow" data-popper-arrow></div>
  </div>
</ComponentDemo>

### Popover position

You can simply change the popover position by changing the **data-placement=" "** attribute with the following values: **_top_**, **_right_**, **_bottom_**, **_left_**.
