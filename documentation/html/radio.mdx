# Tailwind CSS Radio

Our Tailwind CSS radio button component will let your users choose only one of a predefined set of mutually exclusive options. Radio buttons should be used instead of checkboxes if only one item can be selected from a list.
<br/>
Use the following example to create simple radio buttons for your projects.
<br/>

<ComponentCard id="radio">
<div className="text-left">
  <div className="form-check flex mb-4">
    <input className="radio opacity-0 cursor-pointer absolute z-10" type="radio" id="flexRadioDefault1" />
    <div className="form-check-input"></div>
    <label className="custom-control-label my-auto cursor-pointer" for="flexRadioDefault1">Default radio</label >
  </div>
  <div className="form-check flex">
    <input className="radio opacity-0 cursor-pointer absolute z-10" type="radio" id="flexRadioDefault2" checked />
    <div className="form-check-input"></div>
    <label className="custom-control-label my-auto cursor-pointer" for="flexRadioDefault2">Default checked radio</label>
  </div>
</div>
</ComponentCard>
