# Tailwind CSS Switch

Use our Tailwind CSS switch component to let users adjust settings on/off. The option that the switch controls, as well as the state it's in, should be made clear from the corresponding inline label.

Use the following example to create a simple and easy-to-use switch component for your Tailwind CSS project.

<br />

<ComponentPreview id="switch">
  <div className="form-check form-switch mb-4 flex items-center">
    <input
      className="absolute z-10 h-5 w-8 cursor-pointer opacity-0"
      data-attribute="toggle"
      id="remember"
    />
    <div className="form-check-input"></div>
    <label
      className="form-check-label mb-0 ml-2 cursor-pointer select-none"
      for="remember"
    >
      Remember me
    </label>
  </div>
</ComponentPreview>
