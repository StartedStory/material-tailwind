# Tailwind CSS Switch

Get started on your web projects with our Tailwind CSS textarea which provides contextual feedback messages for typical user actions.

<br />

### Input Outline

Use this example to create outline textarea components.

<br />
<ComponentCard id="textarea-outline">
  <div className="input-group input-group-outline mb-6">
    <label className="form-label">Textarea Outline</label>
    <textarea type="text" className="form-control" rows="7"></textarea>
  </div>
</ComponentCard>

```html showLineNumbers
<div class="input-group input-group-outline mb-6">
  <label class="form-label">Textarea Outline</label>
  <textarea type="text" class="form-control" rows="7"></textarea>
</div>
```
