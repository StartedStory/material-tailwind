  Accordion,

  AccordionBody

  const [open, setOpen] = useState(0);

## Accordion Props

<span id="accordion-props" className="scroll-mt-48" />
The following props are available for the accordion component. These are the custom
props that come with we've added for the accordion component and you can use all
the other native props as well.

| Attribute   | Type                       | Description                        | Default                               |
| ----------- | -------------------------- | ---------------------------------- | ------------------------------------- |
| `open`      | <Code small>boolean</Code> | Open accordion collapse            | No default value its a required prop. |
| `icon`      | <Code small>node</Code>    | Change accordion collapse end icon | <Code small>false</Code>              |
| `animate`   | [Animate](#types)          | Change accordion body animation    | <Code small>undefined</Code>          |
| `disabled`  | <Code small>boolean</Code> | Disable the accordion              | <Code small>false</Code>              |
| `className` | <Code small>string</Code>  | Add custom className for accordion | `''`                                  |
| `children`  | <Code small>node</Code>    | Add content for accordion          | No default value its a required prop. |
