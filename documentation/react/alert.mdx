## Alert Dismissible

The alert component is a dismissible component that you can control it using the <Code>dismissible</Code> and <Code>show</Code> props.

<State initialState={{ show: true }}>
  <ComponentCard id="alert-dismissible">
    <Observe>
      {({ setState, show }) => (
        <>
          {!show && (
            <Button
              variant="gradient"
              className="absolute"
              onClick={() => setState((s) => ({ show: true }))}
            >
              Show Alert
            </Button>
          )}
          <Alert
            variant="gradient"
            show={show}
            dismissible={{
              onClose: () => setState((s) => ({ show: false }))
            }}
          >
            A dismissible alert for showing message.
          </Alert>
        </>
      )}
    </Observe>
  </ComponentCard>
</State>

```jsx showLineNumbers {5, 8-26}
import { useState, Fragment } from "react";
import { Alert } from "@material-tailwind/react";

export default function Colors() {
  const [show, setShow] = useState(true);

  return (
    <Fragment>
      {!show && (
        <Button
          variant="gradient"
          className="absolute"
          onClick={() => setShow(true)}
        >
          Show Alert
        </Button>
      )}
      <Alert
        show={show}
        dismissible={{
          onClose: () => setShow(false)
        }}
      >
        A dismissible alert for showing message.
      </Alert>
    </Fragment>
  );
}
```

## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the button component. These are the custom
props that come with we've added for the alert component and you can use all the
other native props as well.

| Attribute     | Type                       | Description                                           | Default                               |
| ------------- | -------------------------- | ----------------------------------------------------- | ------------------------------------- |
| `variant`     | [Variant](#types)          | Change alert variant                                  | <Code small>filled</Code>             |
| `color`       | [Color](#types)            | Change alert color                                    | <Code small>blue</Code>               |
| `dismissible` | [Dismissible](#types)      | Make the alert dismissible                            | <Code small>undefined</Code>          |
| `animate`     | [Animate](#types)          | Controls animation effect of alert                    | <Code small>undefined</Code>          |
| `show`        | <Code small>boolean</Code> | Used to set the alert visibility when its dismissible | <Code small>true</Code>               |
| `icon`        | <Code small>node</Code>    | Used to set an icon at the beginning of alert         | <Code small>undefined</Code>          |
| `className`   | <Code small>string</Code>  | Used to add custom classes for alert                  | `''`                                  |
| `children`    | <Code small>node</Code>    | Used to add the content of the alert                  | No default value its a required prop. |
