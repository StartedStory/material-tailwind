# Tailwind CSS Avatar - React

Use our Tailwind CSS avatar component to portray people or objects in your web projects. The Avatar can be used as a visual identifier for a user profile on your website.

See below our beautiful avatar example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors so you can adapt it easily to your needs.

<br />

<ComponentCard id="avatar">
  <Avatar src="/img/face-2.jpg" />
</ComponentCard>

```jsx showLineNumbers {1, 4}
import { Avatar } from "@material-tailwind/react";

export default function Example() {
  return <Avatar src="/img/face-2.jpg" />;
}
```

---

## Avatar Variants

The avatar component comes with 2 different variants that you can change it using the <Code>variant</Code> prop.

<ComponentCard id="avatar-variants">
  <div className="flex gap-4">
    <Avatar src="/img/face-2.jpg" />
    <Avatar src="/img/face-2.jpg" variant="circular" />
  </div>
</ComponentCard>

```jsx showLineNumbers {6-7}
import { Avatar } from "@material-tailwind/react";

export default function Variants() {
  return (
    <div className="flex gap-4">
      <Avatar src="/img/face-2.jpg" />
      <Avatar src="/img/face-2.jpg" variant="circular" />
    </div>
  );
}
```

## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the avatar component. These are the custom
props that come with we've added for the avatar component and you can use all the
other native img props as well.

| Attribute   | Type                      | Description                     | Default                    |
| ----------- | ------------------------- | ------------------------------- | -------------------------- |
| `variant`   | [Variant](#types)         | Change avatar variant           | <Code small>rounded</Code> |
| `size`      | [Size](#types)            | Change avatar size              | <Code small>md</Code>      |
| `className` | <Code small>string</Code> | Add custom className for avatar | `''`                       |

---

## Types

<span id="types" className="scroll-mt-48" />

```ts
type variant = "rounded" | "circular";
```

```ts
type size = "xs" | "sm" | "md" | "lg" | "xl" | "xxl";
```
