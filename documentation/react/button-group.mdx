# Tailwind CSS Button Group - React

Use our <Code>ButtonGroup</Code> based on Tailwind CSS for actions.

<Code>ButtonGroup</Code> is an essential element of web design. Basically,
<Code>ButtonGroup</Code> is stack of buttons. They help users navigate our websites
or apps and drive them to a particular action.

See below our <Code>ButtonGroup</Code> example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors, so you can adapt it easily to your needs.

<br />

<CodePreview id="button-group" component={
  <ButtonGroup>
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
}>
```jsx
import { ButtonGroup, Button } from "@material-tailwind/react";

export default function Example() {
  return (
    <ButtonGroup>
      <Button>One</Button>
      <Button>Two</Button>
      <Button>Three</Button>
    </ButtonGroup>
  );
}
```
</CodePreview>

## Block Level Button Group

A <Code>ButtonGroup</Code> could be a block level component as well that get's all the available space in a row. You can render a <Code>ButtonGroup</Code> as a block level element using the <Code>fullWidth</Code> prop.

<CodePreview id="block-level-button-group" component={
  <ButtonGroup fullWidth={true}>
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
  </ButtonGroup>
}>
```jsx
import { ButtonGroup, Button } from "@material-tailwind/react";

export default function Example() {
  return (
    <ButtonGroup fullWidth={true}>
      <Button>One</Button>
      <Button>Two</Button>
      <Button>Three</Button>
    </ButtonGroup>
  )
}
```
</CodePreview>
