# Tailwind CSS Button - React

Use our <Code>Button</Code> based on Tailwind CSS for actions in forms, dialogues, and more.

<Code>Button</Code> is an essential element of web design. Basically,
<Code>Button</Code> is styled links that grab the user's attention. They help users
navigate our websites or apps and drive them to a particular action like submitting
a contact form or placing an order as easy as possible.

See below our <Code>Button</Code> example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors, so you can adapt it easily to your needs.

<br />

<CodePreview id="button" link="button#button" component={<Button>Button</Button>}>
```jsx
import { Button } from "@material-tailwind/react";

export default function Example() {
  return <Button>Button</Button>;
}
```
</CodePreview>

<DocsTitle href="button-variants">
## Button Variants
</DocsTitle>

The <Code>Button</Code> component comes with 4 different variants that you can change it using the <Code>variant</Code> prop.

<CodePreview id="button-variants" link="button#button-variants" component={
  <div className="flex w-max gap-4">
    <Button>filled</Button>
    <Button variant="gradient">gradient</Button>
    <Button variant="outlined">outlined</Button>
    <Button variant="text">text</Button>
  </div>
}>
```jsx
import { Button } from "@material-tailwind/react";

export default function Example() {
  return (
    <div className="flex w-max gap-4">
      <Button variant="filled">filled</Button>
      <Button variant="gradient">gradient</Button>
      <Button variant="outlined">outlined</Button>
      <Button variant="text">text</Button>
    </div>
  );
}
```
</CodePreview>

<DocsTitle href="button-with-icon">
## Button with Icon
</DocsTitle>

You can use any type of icons inside the button, in the below example we've used the <a href="https://heroicons.com/?ref=material-tailwind" target="_blank" className="hover:text-blue-500 transition-colors font-medium">@heroicons</a>.

<CodePreview id="button-with-icon" link="button#button-with-icon" component={
  <div className="flex items-center gap-4">
    <Button className="flex items-center gap-3">
      <OutlineIcons.BookmarkIcon strokeWidth={2} className="h-5 w-5" /> Add to Bookmark
    </Button>
    <Button variant="gradient" className="flex items-center gap-3">
      <OutlineIcons.CloudArrowUpIcon strokeWidth={2} className="h-5 w-5" /> Upload Files
    </Button>
    <Button variant="outlined" className="flex items-center gap-3">
      Refresh
      <OutlineIcons.ArrowPathIcon strokeWidth={2} className="h-5 w-5" />
    </Button>
    <Button variant="text" className="flex items-center gap-2">
      Read More <OutlineIcons.ArrowLongRightIcon strokeWidth={2} className="h-5 w-5" />
    </Button>
  </div>
}>

import {
  CloudArrowUpIcon,

  BookmarkIcon,

export default function Day01() {
  return (

        <BookmarkIcon strokeWidth={2} className="h-5 w-5" /> Add to Bookmark

      <Button variant="gradient" className="flex items-center gap-3">
