## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the checkbox component. These are the custom
props that come with we've added for the checkbox component and you can use all the
other native input props as well.

| Attribute        | Type                       | Description                                     | Default                      |
| ---------------- | -------------------------- | ----------------------------------------------- | ---------------------------- |
| `color`          | [Color](#types)            | Change checkbox color                           | <Code small>blue</Code>      |
| `label`          | <Code small>string</Code>  | Used to add a label for the checkbox            | `''`                         |
| `icon`           | <Code small>node</Code>    | Used to change checkbox checked icon            | <Code small>undefined</Code> |
| `ripple`         | <Code small>boolean</Code> | Controls the ripple effect for checkbox         | <Code small>true</Code>      |
| `className`      | <Code small>string</Code>  | Used to add custom classes for button           | `''`                         |
| `containerProps` | <Code small>Object</Code>  | Used to add props for container of the checkbox | <Code small>undefined</Code> |
| `labelProps`     | <Code small>Object</Code>  | Used to add props for label of the checkbox     | <Code small>undefined</Code> |
