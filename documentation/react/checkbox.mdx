# Tailwind CSS Checkbox - React

Use our Tailwind CSS checkbox to allow the user to select one or more items from a set.

You can use a checkbox for:

• Selecting one or more options from a list
• Presenting a list containing sub-selections
• Turning an item on/off in a desktop environment (If you have a single option, avoid using a checkbox and use an on/off switch instead)

See below our simple checkbox example that you can use in your React and Tailwind CSS projects. The example also comes in different colors, so you can adapt it easily to your needs.

<br />

<ComponentCard id="checkbox">
  <Checkbox defaultChecked />
</ComponentCard>

```jsx showLineNumbers {1, 4}
import { Checkbox } from "@material-tailwind/react";

export default function Example() {
  return <Checkbox defaultChecked />;
}
```

## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the checkbox component. These are the custom
props that come with we've added for the checkbox component and you can use all the
other native input props as well.

| Attribute        | Type                       | Description                             | Default                      |
| ---------------- | -------------------------- | --------------------------------------- | ---------------------------- |
| `color`          | [Color](#types)            | Change checkbox color                   | <Code small>blue</Code>      |
| `label`          | <Code small>string</Code>  | Add label for checkbox                  | `''`                         |
| `icon`           | <Code small>node</Code>    | Change checked icon for checkbox        | <Code small>undefined</Code> |
| `ripple`         | <Code small>boolean</Code> | Add ripple effect for checkbox          | <Code small>true</Code>      |
| `className`      | <Code small>string</Code>  | Add custom className for checkbox       | `''`                         |
| `containerProps` | <Code small>Object</Code>  | Add custom props for checkbox container | <Code small>undefined</Code> |
| `labelProps`     | <Code small>Object</Code>  | Add custom props for checkbox label     | <Code small>undefined</Code> |
