# Tailwind CSS Chip - React

Get started on your web projects with our Tailwind CSS <Code>Chip</Code> which is a compact elements that represent an input, attribute, or action. This element appears dynamically as a group of multiple interactive elements and allows users to enter information, make selections, filter content, or trigger actions.

See below our simple <Code>Chip</Code> component example that you can use for your Tailwind CSS and React project. The example also comes in different styles and colors, so you can adapt it easily to your needs.

<br />

<CodePreview id="chip" link="chip#chip" component={<Chip value="chip" />}>
```jsx
import { Chip } from "@material-tailwind/react";

export default function Example() {
  return <Chip value="chip" />;
}
```
</CodePreview>

## Chip Dismissible

The <Code>Chip</Code> component is a dismissible component that you can control it using the <Code>dismissible</Code> and <Code>show</Code> props.

<CodePreview id="chip-dismissible" link="chip#chip-dismissible" component={
  <State initialState={{ open: true }}>
    <Observe>
      {({ setState, open }) => (
        <>
          {!open && (
            <Button
              className="absolute"
              onClick={() => setState((s) => ({ open: true }))}
            >
              Show Chip
            </Button>
          )}
          <Chip
            open={open}
            value="Dismissible"
            onClose={() => setState((s) => ({ open: false }))}
          />
        </>
      )}
    </Observe>
  </State>
}>
```jsx
import { useState, Fragment } from "react";
import { Chip, Button } from "@material-tailwind/react";

export default function Example() {
  const [open, setOpen] = useState(true);

return (

<Fragment>
  {!open && (
    <Button className="absolute" onClick={() => setOpen(true)}>
      Show Chip
    </Button>
  )}
  <Chip open={open} value="Dismissible" onClose={() => setOpen(false)} />
</Fragment>
); }

```
</CodePreview>

## Custom Chip Animation

You can modify the open/close state animation for <Code>Chip</Code> component using the <Code>animate</Code> prop.

<CodePreview id="custom-chip-animation" component={
  <State initialState={{ open: true }}>
    <Observe>
      {({ setState, open }) => (
        <>
          {!open && (
            <Button
              className="absolute"
              onClick={() => setState((s) => ({ open: true }))}
            >
              Show Chip
            </Button>
          )}
          <Chip
            open={open}
            animate={{
              mount: {
                y: 0,
              },
              unmount: {
                y: 50,
              },
            }}
            value="Custom Animation"
            onClose={() => setState((s) => ({ open: false }))}
          />
        </>
      )}
    </Observe>
  </State>
}>
```jsx
import { useState, Fragment } from "react";
import { Chip, Button } from "@material-tailwind/react";

export default function Example() {
  const [open, setOpen] = useState(true);

  return (
    <Fragment>
      {!open && (
        <Button
          className="absolute"
          onClick={() => setOpen(true)}
        >
          Show Chip
        </Button>
      )}
      <Chip
        open={open}
        animate={{
          mount: { y: 0 },
          unmount: { y: 50 },
        }}
        value="Custom Animation"
        onClose={() => setOpen(false)}
      />
    </Fragment>
  );
}
```
</CodePreview>
