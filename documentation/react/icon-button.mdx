# Tailwind CSS Icon Button - React

Use this example to create user-friendly buttons with icon for your Tailwind CSS and React project.

<br />

<CodePreview id="icon-button" link="icon-button#icon-button" component={
  <IconButton>
    <i className="fas fa-heart" />
  </IconButton>
}>
```jsx
import { IconButton } from "@material-tailwind/react";

export default function Example() {
  return (
    <IconButton>
      <i className="fas fa-heart" />
    </IconButton>
  );
}
```
</CodePreview>

<DocsTitle href="icon-button-variants">
## Icon Button Variants
</DocsTitle>

The <Code>IconButton</Code> component comes with 4 different variants that you can change it using the <Code>variant</Code> prop.

<CodePreview id="icon-button-variants" link="icon-button#icon-button-variants" component={

    <IconButton>

    <IconButton variant="gradient">

    </IconButton>
    <IconButton variant="outlined">
      <i className="fas fa-heart" />
    </IconButton>
    <IconButton variant="text">
      <i className="fas fa-heart" />
    </IconButton>
  </div>
}>
```jsx
import { IconButton } from "@material-tailwind/react";

export default function Example() {

      <IconButton>
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton variant="gradient">
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton variant="outlined">
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton variant="text">
        <i className="fas fa-heart" />
      </IconButton>
    </div>
  );
}
```
</CodePreview>

</DocsTitle>

      <i className="fas fa-heart" />

        <i className="fas fa-heart" />

        <i className="fas fa-heart" />
