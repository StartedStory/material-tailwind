# Tailwind CSS Icon Button - React

Use this example to create user-friendly buttons with icon for your Tailwind CSS and React project.

<br />

<ComponentCard id="icon-button">
  <IconButton>
    <i className="fas fa-heart" />

```jsx showLineNumbers
import { IconButton } from "@material-tailwind/react";
```

## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the icon button component. These are the custom
props that come with we've added for the icon button component and you can use all
the other native button props as well.

| Attribute   | Type                       | Description                          | Default                               |
| ----------- | -------------------------- | ------------------------------------ | ------------------------------------- |
| `variant`   | [Variant](#types)          | Change icon button variant           | <Code small>filled</Code>             |
| `size`      | [Size](#types)             | Change icon button size              | <Code small>md</Code>                 |
| `color`     | [Color](#types)            | Change icon button color             | <Code small>blue</Code>               |
| `ripple`    | <Code small>boolean</Code> | Add ripple effect for icon button    | <Code small>true</Code>               |
| `className` | <Code small>string</Code>  | Add custom className for icon button | `''`                                  |
| `children`  | <Code small>node</Code>    | Add content for icon button          | No default value its a required prop. |
