# Tailwind CSS Image - React

Use our Tailwind CSS <Code>img</Code> example to portray people or objects in your web projects. The <Code>img</Code> can be used to display media content on your website.

See below our beautiful <Code>img</Code> example that you can use in your Tailwind CSS and React project. The example also comes in different styles so you can adapt it easily to your needs.

<br />

<CodePreview id="img" link="img#img" component={<ImgExamples.DefaultImg />}>
```jsx

  return (
    <img
      className="h-full w-full"
```

## Image with Caption

<CodePreview id="img-with-caption" link="img#img-with-caption" className="mt-4" component={<ImgExamples.ImgWithCaption />}>
```jsx
import { Typography } from "@material-tailwind/react";

export default function Example() {
  return (
    <figure>
      <img
        className="h-full w-full rounded-lg"
        src="https://images.unsplash.com/photo-1682407186023-12c70a4a35e0?ixlib=rb-4.0.3&ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&auto=format&fit=crop&w=2832&q=80"
        alt="nature image"
      />
      <Typography as="caption" variant="small" className="mt-2 text-center font-normal">
        Image caption
      </Typography>
    </figure>
  );
}
```
</CodePreview>
