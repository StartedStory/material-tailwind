## Input Variants

The input component comes with 3 different variants that you can change it using the <Code>variant</Code> prop.

<ComponentCard id="input-variants">
  <div className="flex items-end gap-4 w-full">
    <Input variant="outlined" label="Outlined" />
    <Input variant="standard" label="Standard" />
    <Input variant="static" label="Static" placeholder="Static" />
  </div>
</ComponentCard>

```jsx showLineNumbers {6-8}
import { Input } from "@material-tailwind/react";

export default function Variants() {
  return (
    <div className="flex items-end gap-4 w-full">
      <Input variant="outlined" label="Outlined" />
      <Input variant="standard" label="Standard" />
      <Input variant="static" label="Static" placeholder="Static" />
    </div>
  );
}
```

## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the checkbox component. These are the custom
props that come with we've added for the checkbox component and you can use all the
other native input props as well.

| Attribute    | Type                       | Description                      | Default                      |
| ------------ | -------------------------- | -------------------------------- | ---------------------------- |
| `variant`    | [Variant](#types)          | Change input variant             | <Code small>outlined</Code>  |
| `size`       | [Size](#types)             | Change input size                | <Code small>md</Code>        |
| `color`      | [Color](#types)            | Change input color               | <Code small>blue</Code>      |
| `label`      | <Code small>string</Code>  | Add label for input              | `''`                         |
| `error`      | <Code small>boolean</Code> | Change input state to error      | <Code small>false</Code>     |
| `success`    | <Code small>boolean</Code> | Change input state to success    | <Code small>false</Code>     |
| `icon`       | <Code small>node</Code>    | Add icon for the input           | <Code small>undefined</Code> |
| `labelProps` | <Code small>Object</Code>  | Add custom props for input label | <Code small>undefined</Code> |
| `className`  | <Code small>string</Code>  | Add custom className for input   | `''`                         |
