# Tailwind CSS Input - React

Easily create inputs with different statuses and sizes using our components based on Tailwind CSS and React.

Input fields are an essential user interface design element, providing users with the means to enter non-standardized responses.

See below our input component example. It comes in different styles and colors, so you can adapt it easily to your needs.

<br />

<ComponentDemo id="input">
  <div className="w-72">
    <Input label="Username" />
  </div>
</ComponentDemo>

```jsx
import { Input } from "@material-tailwind/react";

export default function Example() {
  return <Input label="Username" />;
}
```

---

## Input Variants

The input component comes with 3 different variants that you can change it using the <Code>variant</Code> prop.

<ComponentDemo id="input-variants">
  <div className="flex w-full items-end gap-4">
    <Input variant="outlined" label="Outlined" />
    <Input variant="standard" label="Standard" />
    <Input variant="static" label="Static" placeholder="Static" />
  </div>
</ComponentDemo>

```jsx
import { Input } from "@material-tailwind/react";

export default function Variants() {
  return (
    <div className="flex w-full items-end gap-4">
      <Input variant="outlined" label="Outlined" />
      <Input variant="standard" label="Standard" />
      <Input variant="static" label="Static" placeholder="Static" />
    </div>
  );
}
```

## Props

<span id="props" className="scroll-mt-48" />
The following props are available for input component. These are the custom props
that come with we've added for the input component and you can use all the other
native input props as well.

| Attribute    | Type                 | Description                      | Default                |
| ------------ | -------------------- | -------------------------------- | ---------------------- |
| `variant`    | [Variant](#types)    | Change input variant             | <Code>outlined</Code>  |
| `size`       | [Size](#types)       | Change input size                | <Code>md</Code>        |
| `color`      | [Color](#types)      | Change input color               | <Code>blue</Code>      |
| `label`      | <Code>string</Code>  | Add label for input              | `''`                   |
| `error`      | <Code>boolean</Code> | Change input state to error      | <Code>false</Code>     |
| `success`    | <Code>boolean</Code> | Change input state to success    | <Code>false</Code>     |
| `icon`       | <Code>node</Code>    | Add icon for the input           | <Code>undefined</Code> |
| `labelProps` | <Code>Object</Code>  | Add custom props for input label | <Code>undefined</Code> |
| `className`  | <Code>string</Code>  | Add custom className for input   | `''`                   |
