# Tailwind CSS List - React

Use our responsive Tailwind CSS vertical <Code>List</Code> anywhere on your web app!

A <Code>List</Code> displays a list of items containing images, icons, and text that represents a specific action.

See below our <Code>List</Code> example that you can use in your Tailwind CSS and React project. The examples comes in different styles, so you can adapt them easily to your needs.

<br />

<CodePreview id="list" component={
  <Card className="w-96">
    <List>
      <ListItem>Inbox</ListItem>
      <ListItem>Trash</ListItem>
      <ListItem>Settings</ListItem>
    </List>
  </Card>
}>
```jsx
import { List, ListItem, Card } from "@material-tailwind/react";

export default function Example() {
  return (
    <Card className="w-96">
      <List>
        <ListItem>Inbox</ListItem>
        <ListItem>Trash</ListItem>
        <ListItem>Settings</ListItem>
      </List>
    </Card>
  );
}
```
</CodePreview>

<DocsTitle href="list-with-selected-item">
## List With Selected Item
</DocsTitle>

You can make a list item selected by default using the <Code>selected</Code> prop for the <Code>{`<ListItem />`}</Code> component.

<CodePreview id="list-with-selected-item" component={
  <State initialState={{ selected: 1 }}>
    <Observe>
      {({ setState, selected }) => (
        <Card className="w-96">
          <List>
            <ListItem selected={selected === 1} onClick={() => setState((s) => ({ selected: 1 }))}>Inbox</ListItem>
            <ListItem selected={selected === 2} onClick={() => setState((s) => ({ selected: 2 }))}>Trash</ListItem>
            <ListItem selected={selected === 3} onClick={() => setState((s) => ({ selected: 3 }))}>Settings</ListItem>
          </List>
        </Card>
      )}
    </Observe>
  </State>
}>
```jsx
import React from "react";
import { List, ListItem, Card } from "@material-tailwind/react";

export default function Example() {
  const [selected, setSelected] = React.useState(1);
  const setSelectedItem = (value) => setSelected(value);

  return (
    <Card className="w-96">
      <List>
        <ListItem selected={selected === 1} onClick={() => setSelectedItem(1)}>
          Inbox
        </ListItem>
        <ListItem selected={selected === 2} onClick={() => setSelectedItem(2)}>
          Trash
        </ListItem>
        <ListItem selected={selected === 3} onClick={() => setSelectedItem(3)}>
          Settings
        </ListItem>
      </List>
    </Card>
  );
}

```
</CodePreview>
