# Tailwind CSS Popover - React

Our popover component for Tailwind CSS & React is a small overlay of content that is used to demonstrate secondary information of any component when a user clicks it.

See below our simple popover example that you can use in your web project.

<br />

<ComponentCard id="popover">
  <Popover>
    <PopoverHandler>
      <Button variant="gradient">Show Popover</Button>
    </PopoverHandler>
    <PopoverContent>
      <span>This is a very beautiful popover, show some love.</span>
    </PopoverContent>
  </Popover>
</ComponentCard>

```jsx showLineNumbers
import {
  Popover,
  PopoverHandler,
  PopoverContent,
  Button
} from "@material-tailwind/react";

export default function Example() {
  return (
    <Popover>
      <PopoverHandler>
        <Button variant="gradient">Show Popover</Button>
      </PopoverHandler>
      <PopoverContent>
        This is a very beautiful popover, show some love.
      </PopoverContent>
    </Popover>
  );
}
```

## Popover Props

<span id="popover-props" className="scroll-mt-48" />
The following props are available for the popover component. These are the custom
props that come with we've added for the popover component and you can use all the
other native props as well.

| Attribute   | Type                        | Description                                        | Default                               |
| ----------- | --------------------------- | -------------------------------------------------- | ------------------------------------- |
| `open`      | <Code small>boolean</Code>  | Makes the popover open, when popover is controlled | <Code small>undefined</Code>          |
| `handler`   | <Code small>function</Code> | Change open state for controlled popover           | <Code small>undefined</Code>          |
| `placement` | [Placement](#types)         | Change popover placement                           | <Code small>top</Code>                |
| `offset`    | [Offset](#types)            | Change popover offset from it's handler            | <Code small>5</Code>                  |
| `dismiss`   | [Dismiss](#types)           | Change dismiss listeners when clicking outside     | <Code small>undefined</Code>          |
| `animate`   | [Animate](#types)           | Change popover animation                           | <Code small>undefined</Code>          |
| `children`  | <Code small>node</Code>     | Add content for popover                            | No default value its a required prop. |
