<DocsTitle href="popover">
# Tailwind CSS Popover - React
</DocsTitle>

Our <Code>Popover</Code> component for Tailwind CSS & React is a small overlay of content that is used to demonstrate secondary information of any component when a user clicks it.

See below our simple <Code>Popover</Code> example that you can use in your web project.

<br />

<CodePreview link="popover#popover" component={<PopoverExamples.PopoverDefault />}>
```jsx
import {
  Popover,
  PopoverHandler,
  PopoverContent,
  Button,
} from "@material-tailwind/react";

export function PopoverDefault() {
  return (
    <Popover>
      <PopoverHandler>
        <Button>Show Popover</Button>
      </PopoverHandler>
      <PopoverContent>
        This is a very beautiful popover, show some love.
      </PopoverContent>
    </Popover>
  );
}
```
</CodePreview>

## Popover Custom Animation
</DocsTitle>

You can modify the open/close state animation for <Code>Popover</Code> component using the <Code>animate</Code> prop.

<CodePreview link="popover#popover-custom-animation" component={<PopoverExamples.PopoverCustomAnimation />}>
```jsx
import {
  Popover,
  PopoverHandler,
  PopoverContent,
  Button,
} from "@material-tailwind/react";

export function PopoverCustomAnimation() {
  return (
    <Popover
      animate={{
        mount: { scale: 1, y: 0 },
        unmount: { scale: 0, y: 25 },
      }}
    >
      <PopoverHandler>
        <Button>Show Popover</Button>
      </PopoverHandler>
      <PopoverContent>
        This is a very beautiful popover, show some love.
      </PopoverContent>
    </Popover>
  );
}
```

<DocsTitle href="popover-props">
## Popover Props
</DocsTitle>

The following props are available for popover component. These are the custom props
that we've added for the popover component and you can use all the other
native props as well.

| Attribute   | Type                          | Description                                        | Default                                |
| ----------- | ----------------------------- | -------------------------------------------------- | -------------------------------------- |
| `open`      | <Code>boolean</Code>          | Makes the popover open, when popover is controlled | <Code>undefined</Code>                 |
| `handler`   | <Code>function</Code>         | Change open state for controlled popover           | <Code>undefined</Code>                 |
| `placement` | [Placement](#types-placement) | Change popover placement                           | <Code>top</Code>                       |
| `offset`    | [Offset](#types-offset)       | Change popover offset from it's handler            | <Code>5</Code>                         |
| `dismiss`   | [Dismiss](#types-dismiss)     | Change dismiss listeners when clicking outside     | <Code>undefined</Code>                 |
| `animate`   | [Animate](#types-animate)     | Change popover animation                           | <Code>undefined</Code>                 |
| `children`  | <Code>node</Code>             | Add content for popover                            | No default value it's a required prop. |

<br />
<br />

### For TypeScript Only

```tsx
import type { PopoverProps } from "@material-tailwind/react";
```

<DocsTitle href="popover-theme">
## Popover Theme
</DocsTitle>

Learn how to customize the theme and styles for popover components, the theme object for popover components has two main objects:

**A.** The <Code>defaultProps</Code> object for setting up the default value for props of popover component.<br />
**B.** The <Code>styles</Code> object for customizing the theme and styles of popover component.<br />

You can customize the theme and styles of popover components by adding Tailwind CSS classes as key paired values for objects.

<br />
<br />

<DocsTitle href="popover-theme-object-type">
## Popover Theme Object Type
</DocsTitle>

```ts
interface PopoverStylesType {
  defaultProps: {
    placement: string;
    offset:
      | number
      | {
          mainAxis: number;
          crossAxis: number;
          alignmentAxis: number;
        };
    dismiss: {
      enabled: boolean;
      escapeKey: boolean;
      referencePress: boolean;
      referencePressEvent: 'pointerdown' | 'mousedown' | 'click';
      outsidePress: boolean | ((event: MouseEvent) => boolean);
      outsidePressEvent: 'pointerdown' | 'mousedown' | 'click';
      ancestorScroll: boolean;
      bubbles: boolean | {
          escapeKey: boolean;
          outsidePress: boolean;
      };
    };
    animate: {
      mount: object;
      unmount: object;
    };
    className: string;
  };
  styles: {
    base: object;
  };
}
```

<br />
<br />
