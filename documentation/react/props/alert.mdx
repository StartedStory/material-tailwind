# Tailwind CSS Alert - Props

<span id="alert-props" className="scroll-mt-48" />
The following props are available for the button component. These are the custom
props that come with we've added for the alert component and you can use all the
other native props as well.

| Attribute     | Type                              | Description                        | Default                                |
| ------------- | --------------------------------- | ---------------------------------- | -------------------------------------- |
| `variant`     | [Variant](#types-variant)         | Change alert variant               | <Code small>filled</Code>              |
| `color`       | [Color](#types-color)             | Change alert color                 | <Code small>blue</Code>                |
| `dismissible` | [Dismissible](#types-dismissible) | Makes alert dismissible            | <Code small>undefined</Code>           |
| `animate`     | [Animate](#types-animate)         | Change alert animation             | <Code small>undefined</Code>           |
| `show`        | <Code small>boolean</Code>        | Change alert visibility            | <Code small>true</Code>                |
| `icon`        | <Code small>node</Code>           | Add icon at the beginning of alert | <Code small>undefined</Code>           |
| `className`   | <Code small>string</Code>         | Add custom className for alert     | `''`                                   |
| `children`    | <Code small>node</Code>           | Add content for alert              | No default value it's a required prop. |

<br />
<br />

### For TypeScript Only

```tsx
import type { AlertProps } from "@material-tailwind/react";
```
