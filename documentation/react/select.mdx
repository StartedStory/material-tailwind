# Tailwind CSS Select - React

Use our Tailwind CSS select component to collect user provided information from a list of options.

A select component is a dropdown menu for displaying choices. Use the radio component when there are fewer total options (less than 5).

See below our select component example that you can use in your Tailwind CSS and React project. The example comes in different colors and styles, so you can adapt it easily to your needs.

<br />

<ComponentDemo id="select">
  <div className="w-72">
    <Select label="Select Version">
      <Option>Material Tailwind HTML</Option>
      <Option>Material Tailwind React</Option>
      <Option>Material Tailwind Vue</Option>
      <Option>Material Tailwind Angular</Option>
      <Option>Material Tailwind Svelte</Option>
    </Select>
  </div>
</ComponentDemo>

```jsx showLineNumbers
import { Select, Option } from "@material-tailwind/react";

export default function Example() {
  return (
    <div className="w-72">
      <Select label="Select Version">
        <Option>Material Tailwind HTML</Option>
        <Option>Material Tailwind React</Option>
        <Option>Material Tailwind Vue</Option>
        <Option>Material Tailwind Angular</Option>
        <Option>Material Tailwind Svelte</Option>
      </Select>
    </div>
  );
}
```

## Select Props

<span id="select-props" className="scroll-mt-48" />
The following props are available for the select component. These are the custom
props that come with we've added for the select component and you can use all the
other native props as well.

| Attribute    | Type                        | Description                                     | Default                               |
| ------------ | --------------------------- | ----------------------------------------------- | ------------------------------------- |
| `variant`    | [Variant](#types)           | Change input variant                            | <Code small>outlined</Code>           |
| `size`       | [Size](#types)              | Change input size                               | <Code small>md</Code>                 |
| `color`      | [Color](#types)             | Change input color                              | <Code small>blue</Code>               |
| `label`      | <Code small>string</Code>   | Add label for input                             | `''`                                  |
| `error`      | <Code small>boolean</Code>  | Change input state to error                     | <Code small>false</Code>              |
| `success`    | <Code small>boolean</Code>  | Change input state to success                   | <Code small>false</Code>              |
| `arrow`      | <Code small>node</Code>     | Change select arrow icon                        | <Code small>undefined</Code>          |
| `value`      | <Code small>string</Code>   | Change selected value for select                | <Code small>undefined</Code>          |
| `onChange`   | <Code small>function</Code> | Return selected value when select value changed | <Code small>undefined</Code>          |
| `selected`   | <Code small>function</Code> | Return selected element and it's index          | <Code small>undefined</Code>          |
| `offset`     | [Offset](#types)            | Change select menu offset from it's input       | <Code small>5</Code>                  |
| `dismiss`    | [Dismiss](#types)           | Change dismiss listeners when clicking outside  | <Code small>undefined</Code>          |
| `animate`    | [Animate](#types)           | Change select menu animation                    | <Code small>undefined</Code>          |
| `lockScroll` | <Code small>boolean</Code>  | Lock page scrolling when select menu is opened  | <Code small>false</Code>              |
| `labelProps` | <Code small>Object</Code>   | Add custom props for select label               | <Code small>undefined</Code>          |
| `menuProps`  | <Code small>Object</Code>   | Add custom props for select menu                | <Code small>undefined</Code>          |
| `disabled`   | <Code small>boolean</Code>  | Disable select input                            | <Code small>false</Code>              |
| `className`  | <Code small>string</Code>   | Add custom className for select input           | `''`                                  |
| `children`   | <Code small>node</Code>     | Add content for select                          | No default value its a required prop. |

## Select Option Props

<span id="select-option-props" className="scroll-mt-48" />
The following props are available for the select option component. These are the
custom props that come with we've added for the select option component and you can
use all the other native props as well.

| Attribute   | Type                       | Description                                                           | Default                               |
| ----------- | -------------------------- | --------------------------------------------------------------------- | ------------------------------------- |
| `value`     | <Code small>string</Code>  | Add select option value, it works together with value props of select | <Code small>undefined</Code>          |
| `index`     | <Code small>number</Code>  | Add select option value                                               | <Code small>undefined</Code>          |
| `disabled`  | <Code small>boolean</Code> | Disable select option                                                 | <Code small>false</Code>              |
| `className` | <Code small>string</Code>  | Add custom className for select option                                | `''`                                  |
| `children`  | <Code small>node</Code>    | Add content for select option                                         | No default value its a required prop. |
