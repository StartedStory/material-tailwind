## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the select component. These are the custom
props that come with we've added for the select component and you can use all the
other native props as well.

| Attribute    | Type                        | Description                                         | Default                               |
| ------------ | --------------------------- | --------------------------------------------------- | ------------------------------------- |
| `variant`    | [Variant](#types)           | Change input variant                                | <Code small>outlined</Code>           |
| `size`       | [Size](#types)              | Change input size                                   | <Code small>md</Code>                 |
| `color`      | [Color](#types)             | Change input color                                  | <Code small>blue</Code>               |
| `label`      | <Code small>string</Code>   | Add label for input                                 | `''`                                  |
| `error`      | <Code small>boolean</Code>  | Change input state to error                         | <Code small>false</Code>              |
| `success`    | <Code small>boolean</Code>  | Change input state to success                       | <Code small>false</Code>              |
| `arrow`      | <Code small>node</Code>     | Change select arrow icon                            | <Code small>undefined</Code>          |
| `value`      | <Code small>string</Code>   | Change selected value for select                    | <Code small>undefined</Code>          |
| `onChange`   | <Code small>function</Code> | Return the selected value when select value changed | <Code small>undefined</Code>          |
| `selected`   | <Code small>function</Code> | Return the selected element and it's index          | <Code small>undefined</Code>          |
| `offset`     | [Offset](#types)            | Change the select menu offset from it's input       | <Code small>undefined</Code>          |
| `dismiss`    | [Dismiss](#types)           | Change dismiss listeners when clicking outside      | <Code small>undefined</Code>          |
| `animate`    | [Animate](#types)           | Change select menu animation                        | <Code small>undefined</Code>          |
| `lockScroll` | <Code small>boolean</Code>  | Lock page scrolling when select menu is opened      | <Code small>false</Code>              |
| `labelProps` | <Code small>Object</Code>   | Add custom props for select label                   | <Code small>undefined</Code>          |
| `menuProps`  | <Code small>Object</Code>   | Add custom props for select menu                    | <Code small>undefined</Code>          |
| `disabled`   | <Code small>boolean</Code>  | Disable the select input                            | <Code small>false</Code>              |
| `className`  | <Code small>string</Code>   | Add custom className for input                      | `''`                                  |
| `children`   | <Code small>node</Code>     | Add content for select                              | No default value its a required prop. |
