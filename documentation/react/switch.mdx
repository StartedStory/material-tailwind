# Tailwind CSS Switch - React

Use our Tailwind CSS switch component to let users adjust settings on/off. The option that the switch controls, as well as the state it's in, should be made clear from the corresponding inline label.

See below our example that will help you create simple and easy-to-use switch component for your Tailwind CSS and React project.

<br />

<ComponentPreview id="switch">
  <Switch id="switch-component" defaultChecked />
</ComponentPreview>

```jsx showLineNumbers
import { Switch } from "@material-tailwind/react";

export default function Example() {
  return <Switch defaultChecked />;
}
```

## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the switch component. These are the custom
props that come with we've added for the switch component and you can use all the
other native input props as well.

| Attribute        | Type                       | Description                           | Default                      |
| ---------------- | -------------------------- | ------------------------------------- | ---------------------------- |
| `color`          | [Color](#types)            | Change switch color                   | <Code small>blue</Code>      |
| `label`          | <Code small>string</Code>  | Add label for switch                  | `''`                         |
| `ripple`         | <Code small>boolean</Code> | Add ripple effect for switch          | <Code small>true</Code>      |
| `className`      | <Code small>string</Code>  | Add custom className for switch       | `''`                         |
| `containerProps` | <Code small>Object</Code>  | Add custom props for switch container | <Code small>undefined</Code> |
| `labelProps`     | <Code small>Object</Code>  | Add custom props for switch label     | <Code small>undefined</Code> |
| `circleProps`    | <Code small>Object</Code>  | Add custom props for switch circle    | <Code small>undefined</Code> |
