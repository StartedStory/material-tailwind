# Tailwind CSS Textarea - React

Easily create <Code>Textarea</Code> with different styles using our component based on Tailwind CSS. It is useful when you want to allow users to enter a sizeable amount of free-form text, for example a comment on a review or feedback form.

See below our example that will help you create a simple <Code>Textarea</Code> for your project. It also comes in different colors and styles, so you can adapt it easily to your needs.

<br />

<CodePreview id="textarea" link="textarea#textarea" component={
  <div className="w-96">
    <Textarea label="Message" />
  </div>
}>
```jsx
import { Textarea } from "@material-tailwind/react";

export default function Example() {
  return (
    <div className="w-96">
      <Textarea label="Message" />
    </div>
  );
}
```
</CodePreview>

## Textarea Variants

The <Code>Textarea</Code> component comes with 3 different variants that you can change it using the <Code>variant</Code> prop.

  </div>
}>
```jsx
import { Textarea } from "@material-tailwind/react";
```
