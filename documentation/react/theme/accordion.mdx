# Tailwind CSS Accordion - Theme

<br />
<br />

## Accordion Theme Object Type

<span id="theme-object-type" className="scroll-mt-48" />

```ts
const theme = {
  accordion: {
    defaultProps: {
      icon: ReactNode;
      className: string;
      animate: {
        mount: object;
        unmount: object;
      };
      disabled: boolean;
    };
    styles: {
      base: {
        container: object;
        header: {
          initial: object;
          active: object;
          icon: object;
        };
        body: object;
        disabled: object;
      };
    };
  }
}
```

<br />
<br />

### For TypeScript Only

```tsx
import type { AccordionStylesType } from "@material-tailwind/react";
```

## Accordion Theme Customization

<span id="theme-customization" className="scroll-mt-48" />

```tsx
const theme = {
  accordion: {
    styles: {
      base: {
        header: {
          initial: {
            px: "px-5",
          },
          active: {
            color: "!text-blue-500",
          },
        },
      },
    },
  },
};
```
