# Tailwind CSS Accordion - Theme

Learn how to customize the theme and styles for accordion components, the theme object for accordion components has two main objects:

**A.** The <Code>defaultProps</Code> object for setting up the default value for props of accordion components.<br />
**B.** The <Code>styles</Code> object for customizing the theme and styles of accordion components.<br />

The keys inside the <Code>defaultProps</Code> object are unique and you can only change their values, but for the <Code>styles</Code> object there are two types of keys.

**A.** The unique keys that shouldn't be changed: <Code>base</Code> <Code>container</Code> <Code>header</Code> <Code>initial</Code> <Code>active</Code> <Code>icon</Code> <Code>body</Code> <Code>disabled</Code>.<br />
**B.** The keys that are changeable: all of the keys inside the unique keys object.

You can use Tailwind CSS classes as key paired values inside the unique keys objects.

<br />
<br />

## Accordion Theme Object Type

<span id="theme-object-type" className="scroll-mt-48" />

```ts
const theme = {
  accordion: {
    defaultProps: {
      icon: node;
      className: string;
      animate: {
        mount: object;
        unmount: object;
      };
      disabled: boolean;
    };
    styles: {
      base: {
        container: object;
        header: {
          initial: object;
          active: object;
          icon: object;
        };
        body: object;
        disabled: object;
      };
    };
  }
}
```

## Accordion Theme Customization

<span id="theme-customization" className="scroll-mt-48" />

```tsx
const theme = {
  accordion: {
    styles: {
      base: {
        header: {
          initial: {
            py: "py-4",
          },
          active: {
            color: "!text-blue-grey-900",
          },
        },
      },
    },
  },
};
```
