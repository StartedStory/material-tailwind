# Tailwind CSS Tooltip - React

Customise your web projects with an easy-to-use and responsive Tailwind CSS tooltip!

A tooltip is a small pop-up element that appears while the user moves the mouse pointer over an element. Use it when you want to specify extra information about something when the user moves the mouse pointer over an element.

See below our example that will help you create a simple tooltip for your React project.

<br />
<ComponentDemo id="tooltip">
  <Tooltip content="Material Tailwind">
    <Button variant="gradient">Show Tooltip</Button>
  </Tooltip>
</ComponentDemo>

```jsx showLineNumbers
import { Tooltip, Button } from "@material-tailwind/react";

export default function Example() {
  return (
    <Tooltip content="Material Tailwind">
      <Button variant="gradient">Show Tooltip</Button>
    </Tooltip>
  );
}
```

## Props

<span id="props" className="scroll-mt-48" />
The following props are available for the tooltip component. These are the custom
props that come with we've added for the tooltip component and you can use all the
other native props as well.

| Attribute     | Type                        | Description                                        | Default                                |
| ------------- | --------------------------- | -------------------------------------------------- | -------------------------------------- |
| `open`        | <Code small>boolean</Code>  | Makes the tooltip open, when tooltip is controlled | <Code small>undefined</Code>           |
| `handler`     | <Code small>function</Code> | Change open state for controlled tooltip           | <Code small>undefined</Code>           |
| `content`     | <Code small>node</Code>     | Add content for tooltip                            | <Code small>undefined</Code>           |
| `interactive` | <Code small>boolean</Code>  | Make tooltip interactive                           | <Code small>false</Code>               |
| `placement`   | [Placement](#types)         | Change tooltip placement                           | <Code small>top</Code>                 |
| `offset`      | [Offset](#types)            | Change tooltip offset from it's handler            | <Code small>5</Code>                   |
| `dismiss`     | [Dismiss](#types)           | Change dismiss listeners when clicking outside     | <Code small>undefined</Code>           |
| `animate`     | [Animate](#types)           | Change tooltip animation                           | <Code small>undefined</Code>           |
| `className`   | <Code small>string</Code>   | Add custom className for tooltip                   | `''`                                   |
| `children`    | <Code small>node</Code>     | The element that tooltip will reference to         | No default value it's a required prop. |
